VOU CRIAR UM SISTEMA DE CADASTRO PARA O SETOR MARÍTIMO/PORTUÁRIO CHAMADO "FLEETS FLOW".

**CONTEXTO:**
Temos múltiplos perfis de usuários (Armador Offshore, Terminal Portuário, Praticagem, Seguradora, etc.) que precisam se cadastrar no sistema. Cada perfil tem campos gerais em comum e campos específicos.

**OBJETIVO:**
Criar o banco de dados relacional completo para suportar todos os perfis com seus relacionamentos.

**REQUISITOS:**

1. **TABELA BASE "Empresa":** Campos comuns a todos (Razão Social, CNPJ, Endereço, Responsável Legal, etc.)
2. **TABELA "Perfil":** Cada empresa pode ter um ou mais perfis (ex: mesma empresa pode ser Armador e Contratante)
3. **TABELAS ESPECÍFICAS:** Uma tabela para cada tipo de perfil com seus campos únicos
4. **TABELA "Embarcacao":** Para perfis que possuem frota (Armadores)
5. **TABELA "Certificado_Embarcacao":** Múltiplos certificados por embarcação
6. **TABELA "Tripulante":** Tripulação por embarcação
7. **TABELA "Documento":** Uploads genéricos (Balanço, Certidões Negativas)
8. **TABELA "Integracao_Sistema":** Conexões com sistemas externos (SAP, Totvs, etc.)

**REGAS DE NEGÓCIO:**
- CNPJ único por Empresa
- IMO Number único por Embarcação
- Cada Perfil pertence a uma Empresa
- Certificados têm data de validade
- Suporte a múltiplos uploads de documentos
- Histórico de datas de criação/atualização

**PERFIS SUPORTADOS:**
1. Armador Offshore
2. Armador Portuário Cabotagem  
3. Terminal Portuário
4. Praticagem
5. Armador Fluvial
6. Marítimo / Cruzeiros
7. Armador Portuário Reboque / Lancha
8. Agente Marítimo / Fluvial
9. Contratante
10. Agente Governamental Fiscalizador
11. Certificador Homologador
12. Seguradora

**ENTREGÁVEIS QUE PRECISO:**
1. Script SQL completo com CREATE TABLE, FOREIGN KEYS, INDEXES
2. Diagrama ER em texto (pode ser Mermaid)
3. Views úteis para relatórios
4. Stored Procedures para operações comuns
5. Índices para performance

**OBSERVAÇÕES TÉCNICAS:**
- Usar MySQL/PostgreSQL
- Incluir timestamps para auditoria
- Usar ENUM para campos fixos
- Considerar performance em grandes frotas (milhares de embarcações)
- Incluir validações de data onde aplicável
